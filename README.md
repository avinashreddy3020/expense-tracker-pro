# Expense Tracker 💰

A full-stack expense tracking application to manage daily expenses.

## 🚀 Features
- Add, update, delete expenses
- Track spending
- REST API backend

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB

## 📌 Functionality
- CRUD operations
- Data stored in MongoDB

## 🔧 Setup
1. Clone repo
2. Install dependencies
3. Run server
